# Hotel-Management-website
It is a hotel management website built using Html css and Java script which is made to enhance user experiance using virtal tours and clean UI

## Features:
### Fully customizable
- The components already come with various variants. If you still need to customize more, the code is open source in your components/ui folder.
### Easy code structure
- The code is very easy to understand and easy to use.
### Modernized UI
- All the components are built mainly for modern ui by eradicate the old looking default html components

- ## Available components
- User registration & login
- Browse rooms with images, price, and details
- See the Amenities present
- Works on mobile, tablet, and desktop
- Smooth UI/UX for all screen sizes
